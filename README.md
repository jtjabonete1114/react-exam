
## React Exam

Install dependecies
```
 $ npm install
```

NOTE: Make sure to change apollo client to your runnig API endpoint. <br> 
Exam API:
[https://github.com/kybDev/react-exam-api](https://github.com/kybDev/react-exam-api)

To start project
```
 $ npm start
```

Run Test
```
 $ npm test
```
