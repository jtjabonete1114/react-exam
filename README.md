
## React Exam

Install dependecies
```
 $ npm install
```

NOTE:<br>  
Before staring the app make sure to change the apollo client uri to your runnig API endpoint. <br> 
Exam API:<br> 
[https://github.com/kybDev/react-exam-api](https://github.com/kybDev/react-exam-api)

To start project:
```
 $ npm start
```

Run Test:
```
 $ npm test
```
